# Umbra Engine

Umbra Engine is a custom-built real-time game engine developed in C++, focused on low-level graphics programming, performance, and scalable system design. The project explores the architecture behind modern game engines while maintaining full control over rendering, memory, and core systems.

---

## 🚀 Goals

- Build a game engine from the ground up
- Understand real-time rendering pipelines (Direct3D 12)
- Design clean, modular, and scalable systems
- Achieve high performance through low-level control
- Bridge gameplay systems with graphics programming

---

## 🧠 Core Focus Areas

- Rendering Pipeline (Direct3D 12)
- Mathematics (Vectors, Matrices, Transformations)
- Engine Architecture & System Design
- Memory Management
- Input & Event Systems
- Real-time Performance Optimization

## ⚙️ Current Status

Umbra Engine is currently under active development. Core systems are being built incrementally, starting from foundational components such as the windowing system, game loop, and rendering pipeline.

---

## 🎯 Vision

The goal of Umbra Engine is not just to build a functional engine, but to deeply understand how game engines work internally—combining graphics programming with gameplay systems to create a solid technical foundation for future projects.

---

## 📌 Notes

This project serves as both a learning platform and a long-term engine development effort. Features are implemented step-by-step with a focus on clarity, performance, and maintainability.
